# cpi_inflation
# Consumer Price Index (CPI) Data Analysis – India

## Project Overview
This project analyzes the **All India Consumer Price Index (CPI)** dataset to understand inflation trends across different commodity categories and time periods. The dataset was cleaned, transformed, and analyzed to extract meaningful insights using data analysis techniques.

The goal of this project is to demonstrate **data cleaning, exploratory data analysis (EDA), pivot table analysis, and data visualization skills** using real-world economic data.

---

## Dataset
The dataset used in this project is **All India CPI Index data up to April**.

It contains CPI values across different commodity groups and time periods which help analyze **price changes and inflation trends in India**.

---

## Project Workflow

1. **Data Collection**
   - Imported CPI dataset containing All India Index values.

2. **Data Cleaning**
   - Converted raw dataset into a structured CPI case study dataset.
   - Handled missing values.
   - Removed duplicates and unnecessary columns.
   - Standardized column names for better readability.
   - Corrected data types.

3. **Data Analysis**
   - Performed descriptive statistics.
   - Conducted category-wise CPI comparison.
   - Analyzed CPI trends across time periods.

4. **Advanced Analytics**
   - Created **Pivot Tables** for deeper analysis.
   - Performed **aggregation and comparative analysis**.

5. **Data Visualization**
   - Built multiple charts to interpret CPI patterns and trends.
